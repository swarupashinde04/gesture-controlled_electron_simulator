# ⚡ Gesture Controlled Electric Field Simulator

A real-time physics simulation where hand gestures create an electric field and influence the motion of an electron.

This project combines **Computer Vision + Physics + Python** to demonstrate how technology can be used to visualize scientific concepts interactively.

---

## 🚀 Features

* Detects hand motion using **OpenCV + MediaPipe**
* Creates a virtual electric field based on hand position
* Simulates electron behavior inside the field
* Real-time visualization using **Pygame**
* Beginner-friendly but conceptually advanced

---

## 🛠️ Technologies Used

* Python
* OpenCV
* MediaPipe
* Pygame

---

## 📚 Concept Behind the Project

Electric fields affect the motion of charged particles like electrons.

In this simulation:

👉 Your **hand acts as the electric field source**
👉 The **electron responds to the field**
👉 Movement is shown in real time

This makes physics more visual, interactive, and easier to understand.
